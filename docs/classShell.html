void Shell::main ( )

The main function of the shell.

It will prompt the user for a command; Then it calls the internal 'parse()' method to parse the input line, building a matching Sequence datastructure; Then it calls the internal 'execute()' command to have the commandline executed. This cycle is repeated until end of input or some error occurs.
