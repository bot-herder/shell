void Command::execute ( )

Execute the command.

All words are passed to the new process as the program's argument list.

The first word is taken to be the name of the program to be executed. If the name begins with '/' it is an absolute name. If the name begins with "./" it is relative to the current directory. Otherwise it is to be searched for using the PATH environment variable. (Also see getenv(3), getcwd(3), access(2), execv(3)).
