Detailed Description

This file defines various macro's that can be used as checks in your program.


If NDEBUG has been defined the assertions will be totally disabled for that translation unit.

Note:
It relies on the __FILE__, __LINE__ and __PRETTY_FUNCTION__ compiler magics to actually identify lines of source code.
If your compiler does not provide the __PRETTY_FUNCTION__ magic, simply #define it as 0 and the 'asserts' function will ignore that parameter.
